# 豆包图片任务监控插件

基于 Chrome/Edge Manifest V3 的侧边栏扩展，用于统一监控多个豆包页面的生图任务，并支持批量发送提示词。

## 核心能力

- 侧边栏集中展示所有豆包标签页任务状态
- 支持单标签发送与“发送全部可发送标签页”
- 自动跳过生成中、已达上限、离线或不可输入页面
- 支持预设提示词、快速切换与持久化存储
- 实时同步状态并高亮当前激活标签页

## 状态判定规则

- 生成中：页面存在运行中任务标记
- 已结束：页面不存在运行中任务标记
- 已达上限：出现登录解锁弹窗后，批量发送自动跳过

## 安装

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 开启开发者模式
3. 点击“加载已解压的扩展程序”并选择本目录
4. 打开豆包页面后，点击扩展图标进入侧边栏

## 使用流程

1. 在侧边栏输入或选择预设提示词
2. 点击“发送全部”或单任务发送
3. 观察任务卡状态、图片数与统计信息
4. 必要时点击“刷新”重新拉取状态

## 数据存储

使用 `chrome.storage.local` 保存：

- `promptText`：当前输入提示词
- `promptPresets`：预设提示词列表
- `autoRefreshSeconds`：刷新配置项（当前版本保留）

## 页面元素识别

- 输入框：页面主输入区域
- 发送按钮：页面发送操作按钮
- 运行中标记：用于判断任务是否仍在生成
- 上限弹窗：用于判断当前页面是否达到限制

## 常见问题

- 显示“未连接”：刷新豆包标签页后在侧边栏点“刷新”
- 后台标签页发送失败：扩展会自动激活该页重试并恢复原标签页
